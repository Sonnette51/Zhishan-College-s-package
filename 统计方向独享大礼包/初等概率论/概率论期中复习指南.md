## 第一部分：概率空间的基础框架

### 1.1 三元组 (Ω, F, P) 的严格定义

**样本空间 Ω**：一次随机试验所有可能结果的集合。定义时必须满足**完备性**（不遗漏）和**互斥性**（每次试验恰好对应一个 ω ∈ Ω）。

**事件域 F（σ-代数）**：Ω 的某些子集构成的集族，需满足三条公理：

1. Ω ∈ F（全集属于事件域）
2. 若 A ∈ F，则 Aᶜ ∈ F（对补运算封闭）
3. 若 A₁, A₂, ... ∈ F，则 ∪ᵢ₌₁^∞ Aᵢ ∈ F（对可列并封闭）

由这三条可推出：∅ ∈ F，对可列交也封闭。

**概率测度 P**：F → [0,1] 的集函数，满足三条公理：

1. 非负性：P(A) ≥ 0
2. 规范性：P(Ω) = 1
3. 可列可加性：若 A₁, A₂, ... 两两互斥，则 P(∪ᵢ₌₁^∞ Aᵢ) = Σᵢ₌₁^∞ P(Aᵢ)

### 1.2 由事件生成的 σ-代数（真题第八题第(1)问）

**题目**：Ω = [0,1]，A₁ = (1/3, 1/3⁰] = (1/3, 1]，写出由 {A₁} 生成的事件域 F。

**解题过程**：

对于单个事件 A，生成的最小 σ-代数包含 A 本身，然后逐步取补、取并，直到封闭：

- A₁ = (1/3, 1] ∈ F
- A₁ᶜ = [0, 1/3] ∈ F（取补）
- Ω = A₁ ∪ A₁ᶜ = [0,1] ∈ F
- ∅ = A₁ ∩ A₁ᶜ ∈ F

不再产生新集合，故 **F = {∅, (1/3, 1], [0, 1/3], [0, 1]}**，共 4 个元素。

> **一般规律**：由 n 个"独立"事件生成的 σ-代数最多有 2^(2ⁿ) 个元素。单个事件生成 2² = 4 个。

### 1.3 事件的极限（真题第八题第(2)(3)问）

**定义回顾**：

- **上极限**：lim sup Aₙ = ∩ₙ₌₁^∞ ∪ₖ₌ₙ^∞ Aₖ，含义是"无穷多个 Aₙ 发生"（属于无穷多个 Aₙ 的 ω）
- **下极限**：lim inf Aₙ = ∪ₙ₌₁^∞ ∩ₖ₌ₙ^∞ Aₖ，含义是"至多有限个 Aₙ 不发生"（从某个 n 起所有 Aₙ 都发生）

**题目(2)**：Aₙ = (1/3ⁿ, 1/3ⁿ⁻¹]，P 定义为区间长度，求 P(lim_{n→∞} ∪ᵢ₌₁ⁿ Aᵢ)。

**解题过程**：

令 Bₙ = ∪ᵢ₌₁ⁿ Aᵢ。注意 Aᵢ 两两不相交（因为 (1/3ⁱ, 1/3ⁱ⁻¹] 互不重叠），所以：

$$B_n = \bigcup_{i=1}^n A_i = (1/3^n,\, 1]$$

因此 Bₙ 是单调递增序列（Bₙ ↑），其极限为：

$$\lim_{n\to\infty} B_n = \bigcup_{n=1}^\infty B_n = (0, 1]$$

由概率的连续性：

$$P\left(\lim_{n\to\infty} B_n\right) = \lim_{n\to\infty} P(B_n) = \lim_{n\to\infty} \left(1 - \frac{1}{3^n}\right) = 1$$

**题目(3)**：求 lim sup Aₙ。

**解题过程**：

$$\lim\sup A_n = \bigcap_{n=1}^\infty \bigcup_{k=n}^\infty A_k = \bigcap_{n=1}^\infty (0,\, 1/3^{n-1}]$$

随着 n → ∞，1/3ⁿ⁻¹ → 0，这些区间的交集缩小为 {0} 或 ∅。

由于每个集合形如 (0, 1/3ⁿ⁻¹]（不包含 0），所以交集中没有任何点（对任意正数 ε，当 n 足够大时 1/3ⁿ⁻¹ < ε）。

但注意 ω = 0 不在任何 (0, 1/3ⁿ⁻¹] 中，故 **lim sup Aₙ = ∅**。

> **直观理解**：Aₙ 的"位置"不断左移趋近于 0 且"宽度"趋于 0，不可能有任何点属于无穷多个 Aₙ。

## 第二部分：概率的性质与证明

### 2.1 由公理推导的核心性质

以下每一条都可以从三条公理出发严格证明（考试可能要求证明过程）：

**有限可加性**：若 A₁, ..., Aₙ 两两互斥，则 P(∪ᵢ₌₁ⁿ Aᵢ) = Σᵢ₌₁ⁿ P(Aᵢ)。  
*证明*：令 Aₙ₊₁ = Aₙ₊₂ = ... = ∅，由可列可加性即得。

**单调性**：A ⊂ B ⟹ P(A) ≤ P(B)。  
*证明*：B = A ∪ (B\A)，两者互斥，故 P(B) = P(A) + P(B\A) ≥ P(A)。

**容斥原理**：P(A∪B) = P(A) + P(B) − P(A∩B)。  
*证明*：A∪B = A ∪ (B\A)，而 P(B\A) = P(B) − P(A∩B)。

**Boole 不等式（次可加性）**：P(∪ᵢ₌₁^∞ Aᵢ) ≤ Σᵢ₌₁^∞ P(Aᵢ)。

**连续性（极限定理）**：

- Aₙ ↑ A（单调递增）⟹ P(Aₙ) → P(A)
- Aₙ ↓ A（单调递减）⟹ P(Aₙ) → P(A)

*证明思路*：设 Aₙ ↑ A，令 B₁ = A₁, Bₙ = Aₙ \ Aₙ₋₁（n ≥ 2），则 Bₙ 两两互斥且 ∪Bₙ = A。由可列可加性 P(A) = Σ P(Bₙ) = lim Σᵢ₌₁ⁿ P(Bᵢ) = lim P(Aₙ)。

> **证明工具总结**：拆成互斥集合、归纳法、求补——这三招是概率性质证明的核心武器。

### 2.2 判断题经典陷阱详解（真题第一题）

**题1**：若对任意 ω ∈ A，皆有 P({ω}) = 0，则 P(A) = 0。**（假）**

反例：Ω = [0,1]，P 为 Lebesgue 测度（均匀分布），A = [0, 0.5]。  
每个单点 P({ω}) = 0，但 P(A) = 0.5。  
关键：可列可加性只对**可列**个集合成立。当 A 是不可列集合时，P(A) ≠ Σ_{ω∈A} P({ω})，因为不可列求和没有意义。

**题2**：X ~ P(λ)，则 P(X = k+1 | X > k) = P(X = 1)。**（假）**

计算验证：
$$P(X=k+1|X>k) = \frac{P(X=k+1)}{P(X>k)} = \frac{e^{-\lambda}\lambda^{k+1}/(k+1)!}{1 - \sum_{i=0}^{k} e^{-\lambda}\lambda^i/i!}$$

而 P(X=1) = λe^{-λ}。两者一般不相等。  
**本质**：泊松分布**没有**无记忆性。无记忆性是几何分布（离散）和指数分布（连续）的专属性质。

**题3**：X 是连续型随机变量，则 PDF 是连续函数。**（假）**

反例：f(x) = 1/2（0 < x < 1）, f(x) = 1/4（2 < x < 4），其余为 0。  
这是合法的 PDF（积分为 1），但在 x = 0, 1, 2, 4 处不连续。  
**要点**：PDF 只需要非负且积分为 1，不要求自身连续。改变有限个点的值不影响积分。

**题4**：(U,V) 是连续型随机向量，则 P(U = V) = 0。**（真）**

证明：
$$P(U=V) = \iint_{u=v} f(u,v)\, du\, dv = \int_{-\infty}^{\infty} \left[\int_{u=v} f(u,v)\, du \right] dv$$
内层积分在 u = v 这条线上（一维），对 u 来说是零测集上的积分，因此整体为 0。

**题5**：随机变量 X 是一种映射，把每个试验结果映射为一个实数。**（真）**

但要补充：X: Ω → ℝ 必须是**可测映射**，即对任意 Borel 集 B，{ω: X(ω) ∈ B} ∈ F。

**题6**：A∩B ≠ ∅ 则 A, B 不独立。**（假）**

反例：掷骰子，A = {1,2,3,4}（P=2/3），B = {1,2,3}（P=1/2）。A∩B = {1,2,3}，P(A∩B) = 1/2。而 P(A)·P(B) = 2/3 × 1/2 = 1/3 ≠ 1/2，所以这个例子确实不独立。  
换一个：A = {1,2}，B = {1,3,5}，P(A∩B) = P({1}) = 1/6 = P(A)·P(B) = 2/6 × 3/6 = 1/6。✓ 独立且 A∩B ≠ ∅。

**题7**：A, B 独立，则给定 C 下 A, B 也条件独立。**（假）**

经典反例：两次独立掷硬币，A = 第一次正面，B = 第二次正面，C = 两次结果相同。  
P(A) = P(B) = 1/2，P(A∩B) = 1/4 = P(A)P(B)，A, B 独立。  
但 P(A∩B|C) = P({HH})/P({HH,TT}) = (1/4)/(1/2) = 1/2。  
P(A|C)·P(B|C) = (1/2) × (1/2) = 1/4 ≠ 1/2。故条件不独立。

**题8**：已知事件 A 发生，则 P(A) = 1。**（假）**

"A 发生"指的是在某次具体试验中，结果 ω ∈ A。这是一个观测结果。  
P(A) = 1 是概率度量的性质，表示"几乎必然发生"。  
反例：掷骰子得到 1，事件 A = {1}，A 确实发生了，但 P(A) = 1/6。

**题9**：X, Y 都服从标准正态，则 (X,Y) 服从二元正态但不能确定 ρ。**（真）**

边缘分布都是 N(0,1) 只确定了 μ₁, μ₂, σ₁, σ₂。相关系数 ρ = Cov(X,Y)/(σ₁σ₂) 取决于 X, Y 之间的相关结构，需要额外信息。

> **注意**：此题的前提隐含了 (X,Y) 是二元正态的假设。实际上，边缘都是正态并不能保证联合一定是二元正态！如果题目没有给出联合正态的假设，连 (X,Y) 是否服从二元正态都不能确定。

## 第三部分：条件概率与三大法则

### 3.1 条件概率的定义与直觉

$$P(A|B) = \frac{P(A \cap B)}{P(B)}, \quad P(B) > 0$$

**直觉**：已知 B 发生，将样本空间"缩小"到 B，在 B 中看 A 的比例。

### 3.2 三大法则

**乘法法则**：P(A₁∩A₂∩...∩Aₙ) = P(A₁) · P(A₂|A₁) · P(A₃|A₁∩A₂) · ... · P(Aₙ|A₁∩...∩Aₙ₋₁)

**全概率公式**：若 B₁, B₂, ..., Bₙ 构成 Ω 的划分（两两互斥且并为 Ω），则
$$P(A) = \sum_{i=1}^n P(A|B_i) \cdot P(B_i)$$

**Bayes 公式**：
$$P(B_j|A) = \frac{P(A|B_j) \cdot P(B_j)}{\sum_{i=1}^n P(A|B_i) \cdot P(B_i)}$$

> **解题直觉**：全概率 = "由因求果"（已知各原因的概率，求结果的概率）；Bayes = "由果溯因"（观察到结果，反推最可能的原因）。

### 3.3 真题完整解答：疫苗有效性问题（第三题）

**题目**：一人一年感冒次数 ~ P(5)。疫苗对 75% 的人有效（降为 P(3)），对 25% 的人无效。某人接种后一年感冒 2 次，疫苗对他有效的概率？

**解答**：

设 E = 疫苗有效，A = 一年感冒 2 次。

先验概率：P(E) = 0.75，P(Eᶜ) = 0.25。

条件概率（似然）：

$$P(A|E) = \frac{e^{-3} \cdot 3^2}{2!} = \frac{9e^{-3}}{2} \approx \frac{9 \times 0.0498}{2} \approx 0.2240$$

$$P(A|E^c) = \frac{e^{-5} \cdot 5^2}{2!} = \frac{25e^{-5}}{2} \approx \frac{25 \times 0.00674}{2} \approx 0.0842$$

由 Bayes 公式：

$$P(E|A) = \frac{P(A|E) \cdot P(E)}{P(A|E) \cdot P(E) + P(A|E^c) \cdot P(E^c)}$$

$$= \frac{0.2240 \times 0.75}{0.2240 \times 0.75 + 0.0842 \times 0.25} = \frac{0.1680}{0.1680 + 0.02105} \approx \frac{0.1680}{0.1891} \approx 0.889$$

精确表达：

$$P(E|A) = \frac{9e^{-3} \times 0.75}{9e^{-3} \times 0.75 + 25e^{-5} \times 0.25} = \frac{6.75 e^{-3}}{6.75 e^{-3} + 6.25 e^{-5}} = \frac{6.75}{6.75 + 6.25 e^{-2}}$$

$$= \frac{6.75}{6.75 + 6.25 \times 0.1353} = \frac{6.75}{6.75 + 0.8456} = \frac{6.75}{7.596} \approx 0.889$$

> **Bayes 题解题模板**：① 明确"原因"事件（E 和 Eᶜ 构成划分）；② 写出先验 P(E), P(Eᶜ)；③ 分别计算似然 P(A|E), P(A|Eᶜ)；④ 套 Bayes 公式；⑤ 化简。

## 第四部分：独立性专题

### 4.1 事件独立性的定义层次

| 概念 | 定义 | 需要验证的等式数 |
|------|------|------------------|
| A, B 独立 | P(A∩B) = P(A)·P(B) | 1 个 |
| A, B, C 两两独立 | 任意两个满足独立 | 3 个：P(AB), P(AC), P(BC) |
| A, B, C 相互独立 | 两两独立 + P(A∩B∩C) = P(A)P(B)P(C) | 4 个（3+1） |
| n 个事件相互独立 | 任意 k 个（2 ≤ k ≤ n）的交的概率等于各自概率的乘积 | 2ⁿ − n − 1 个 |

> **两两独立 ⇏ 相互独立**：经典反例是 Bernstein 反例。两次掷硬币，A = 第一次正，B = 第二次正，C = 两次结果不同。可验证两两独立但不相互独立。

### 4.2 独立性 vs 条件独立

- **独立 ⇏ 条件独立**：反例见上面题 7 的分析。
- **条件独立 ⇏ 独立**：同样可以构造反例。

**核心**：独立性和条件独立性是两个互不蕴含的概念，必须独立验证。

### 4.3 随机变量的独立性

X, Y 独立 ⟺ F(x,y) = Fₓ(x) · F_Y(y) 对所有 x, y 成立。

等价条件：
- 离散型：P(X=xᵢ, Y=yⱼ) = P(X=xᵢ) · P(Y=yⱼ) 对所有 i, j 成立
- 连续型：f(x,y) = fₓ(x) · f_Y(y) 几乎处处成立

**独立性的快速判定**：
1. 联合 PDF/PMF 能否写成 g(x)·h(y) 的形式？
2. 定义域是否为矩形区域（x 的范围不依赖 y，反之亦然）？
3. 两个条件**同时满足**才独立。

> **常见陷阱**：f(x,y) = 6xy（0 < x < 1, 0 < y < 1）可以分离且区域是矩形，独立。但 f(x,y) = 2（0 < x < y < 1）虽然常数可分离，但区域是三角形（x < y），**不独立**。

## 第五部分：分布函数与混合型分布

### 5.1 CDF 的性质（必须牢记）

1. **单调不减**：x₁ < x₂ ⟹ F(x₁) ≤ F(x₂)
2. **右连续**：F(x) = F(x⁺) = lim_{t→x⁺} F(t)
3. **极限值**：F(-∞) = 0，F(+∞) = 1
4. **跳跃量 = 点概率**：P(X = a) = F(a) − F(a⁻)

### 5.2 真题完整解答：混合型分布（第二题）

**题目**：

$$F(x) = \begin{cases} 0, & x < 1 \\ \frac{1}{3}x - \frac{1}{3}, & 1 \leq x < 2 \\ \frac{1}{3}, & 2 \leq x < 3 \\ 1, & x \geq 3 \end{cases}$$

**(1) 画图**：

- [x < 1]：F = 0（水平线）
- [1, 2)：从 F(1) = 0 线性增长到 F(2⁻) = 1/3（斜率 1/3 的直线段）
- [2, 3)：F = 1/3（水平线，无跳跃）
- x = 3：跳跃到 F(3) = 1

注意 F 在 x = 1 处连续（F(1) = 0 = F(1⁻)），在 x = 3 处跳跃（F(3) = 1 ≠ F(3⁻) = 1/3）。

**(2) P(X = 2)**：

$$P(X=2) = F(2) - F(2^-) = \frac{1}{3} - \left(\frac{1}{3} \times 2 - \frac{1}{3}\right) = \frac{1}{3} - \frac{1}{3} = 0$$

x = 2 处 CDF 无跳跃，所以该点概率为 0。

**(3) P(X = 3)**：

$$P(X=3) = F(3) - F(3^-) = 1 - \frac{1}{3} = \frac{2}{3}$$

x = 3 处有跳跃，跳跃量即为该点概率。

**(4) P(X ≥ 3/2)**：

$$P(X \geq 3/2) = 1 - P(X < 3/2) = 1 - F(3/2^-)$$

由于 F 在 [1, 2) 上连续，F(3/2⁻) = F(3/2) = 1/3 × 3/2 − 1/3 = 1/2 − 1/3 = 1/6。

$$P(X \geq 3/2) = 1 - \frac{1}{6} = \frac{5}{6}$$

> **关键方法**：混合型分布画图是第一步！跳跃点用 F(a) − F(a⁻) 求概率，连续段用 CDF 值直接计算。注意 P(X ≥ a) = 1 − F(a⁻) 而不是 1 − F(a)。

## 第六部分：常见分布全景

### 6.1 离散型分布

**两点分布 B(1,p)**：X ∈ {0,1}，P(X=1) = p。E(X) = p，Var(X) = p(1−p)。

**二项分布 B(n,p)**：n 次独立 Bernoulli 试验中成功次数。  
P(X=k) = C(n,k) pᵏ(1−p)ⁿ⁻ᵏ，k = 0,1,...,n。  
E(X) = np，Var(X) = np(1−p)。

**几何分布 Geom(p)**：首次成功时的试验次数。  
P(X=k) = (1−p)ᵏ⁻¹p，k = 1,2,...  
E(X) = 1/p，Var(X) = (1−p)/p²。  
**无记忆性**：P(X > m+n | X > m) = P(X > n)。这是几何分布独有的离散无记忆性！

**泊松分布 P(λ)**：单位时间（或空间）内稀有事件发生次数。  
P(X=k) = e⁻λ λᵏ/k!，k = 0,1,2,...  
E(X) = λ，Var(X) = λ。

**泊松近似**：当 n 大、p 小、np = λ 适中时，B(n,p) ≈ P(λ)。

### 6.2 连续型分布

**均匀分布 U(a,b)**：f(x) = 1/(b−a)，a < x < b。  
E(X) = (a+b)/2，Var(X) = (b−a)²/12。

**指数分布 Exp(λ)**：f(x) = λe⁻λˣ，x ≥ 0。  
F(x) = 1 − e⁻λˣ。E(X) = 1/λ，Var(X) = 1/λ²。  
**无记忆性**：P(X > s+t | X > s) = P(X > t)。

**正态分布 N(μ, σ²)**：  
$$f(x) = \frac{1}{\sqrt{2\pi}\sigma} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$$

标准化：Z = (X−μ)/σ ~ N(0,1)，CDF 记为 Φ(z)。  
性质：Φ(−z) = 1 − Φ(z)。

**线性变换**：若 X ~ N(μ, σ²)，则 aX + b ~ N(aμ+b, a²σ²)。

### 6.3 分布间的关系图谱

```
二项分布 B(n,p) --[n大p小]--> 泊松分布 P(np)
二项分布 B(n,p) --[n大]--> 正态分布 N(np, np(1-p))  (中心极限定理)
指数分布 Exp(λ) <--[连续版]--> 几何分布 Geom(p)  (都有无记忆性)
均匀分布 U(0,1) --[Y=-2lnX]--> 指数分布 Exp(1/2)  (真题第四题)
泊松过程中 --> 到达间隔 ~ 指数分布，计数 ~ 泊松分布
```

## 第七部分：随机变量函数的分布

### 7.1 CDF 法（万能方法，必须熟练掌握）

**步骤**：

1. 确定 Y = g(X) 的值域
2. 写出 F_Y(y) = P(Y ≤ y) = P(g(X) ≤ y)
3. 将不等式 g(X) ≤ y 化为 X 的范围（注意 g 的单调性！增函数保号，减函数变号）
4. 用 X 的 CDF 或 PDF 表达
5. 对 y 求导得 f_Y(y)
6. **写清定义域**

### 7.2 真题完整解答：变量函数的分布（第四题）

**(1) X ~ N(1,1)，求 Y = 2X − 1 的 PDF**

**方法一：线性变换性质**（最快）

Y = 2X − 1，X ~ N(1,1)。  
E(Y) = 2·1 − 1 = 1，Var(Y) = 4·1 = 4。  
故 Y ~ N(1, 4)。

$$f_Y(y) = \frac{1}{\sqrt{2\pi} \cdot 2} \exp\left(-\frac{(y-1)^2}{8}\right), \quad -\infty < y < \infty$$

**方法二：CDF 法**（展示完整过程）

$$F_Y(y) = P(2X-1 \leq y) = P\left(X \leq \frac{y+1}{2}\right) = \Phi\left(\frac{(y+1)/2 - 1}{1}\right) = \Phi\left(\frac{y-1}{2}\right)$$

求导：

$$f_Y(y) = \frac{d}{dy}\Phi\left(\frac{y-1}{2}\right) = \frac{1}{2}\varphi\left(\frac{y-1}{2}\right) = \frac{1}{2} \cdot \frac{1}{\sqrt{2\pi}} e^{-(y-1)^2/8}$$

与方法一结果一致。

**(2) X ~ U(0,1)，求 Y = −2lnX 的 PDF**

**Step 1**：Y 的值域。X ∈ (0,1)，−lnX ∈ (0, +∞)，故 Y ∈ (0, +∞)。

**Step 2**：写 CDF。

$$F_Y(y) = P(-2\ln X \leq y) = P\left(\ln X \geq -\frac{y}{2}\right) = P\left(X \geq e^{-y/2}\right)$$

当 y > 0 时，0 < e^{-y/2} < 1，所以：

$$F_Y(y) = 1 - F_X(e^{-y/2}) = 1 - e^{-y/2}$$

**Step 3**：求导。

$$f_Y(y) = \frac{d}{dy}(1 - e^{-y/2}) = \frac{1}{2}e^{-y/2}, \quad y > 0$$

这正是 **Exp(1/2)** 的 PDF（即参数 λ = 1/2 的指数分布）。

> **检验**：∫₀^∞ (1/2)e^{-y/2} dy = 1 ✓

## 第八部分：联合分布、边缘分布与独立性

### 8.1 真题完整解答：联合分布问题（第五题）

**题目**：

$$F(x,y) = \begin{cases} c(1-e^{-x})(2-2e^{-2y}), & x,y \geq 0 \\ 0, & \text{otherwise} \end{cases}$$

**(1) 求常数 c 和联合 PDF**

由 F(+∞, +∞) = 1：

$$c \cdot (1 - 0) \cdot (2 - 0) = 2c = 1 \implies c = \frac{1}{2}$$

联合 PDF：

$$f(x,y) = \frac{\partial^2 F}{\partial x \partial y} = \frac{1}{2} \cdot e^{-x} \cdot 4e^{-2y} = 2e^{-x}e^{-2y}, \quad x > 0,\ y > 0$$

**(2) X, Y 是否独立？**

提取边缘 CDF：
- Fₓ(x) = F(x, +∞) = (1/2)(1 − e⁻ˣ) · 2 = 1 − e⁻ˣ（x ≥ 0）→ X ~ Exp(1)
- F_Y(y) = F(+∞, y) = (1/2) · 1 · (2 − 2e⁻²ʸ) = 1 − e⁻²ʸ（y ≥ 0）→ Y ~ Exp(2)

验证：F(x,y) = (1/2)(1−e⁻ˣ)(2−2e⁻²ʸ) = (1−e⁻ˣ)(1−e⁻²ʸ) = Fₓ(x) · F_Y(y)。✓

**X, Y 相互独立。**

也可从 PDF 验证：f(x,y) = 2e⁻ˣe⁻²ʸ = (e⁻ˣ)(2e⁻²ʸ) = fₓ(x) · f_Y(y)，且定义域 (0,∞)×(0,∞) 是矩形。✓

**(3) X 服从的分布**

X ~ Exp(1)，即参数 λ = 1 的指数分布。

## 第九部分：多项分布与抽样模型

### 9.1 多项分布的定义与性质

n 次独立试验，每次有 k 种结果，概率为 p₁, ..., pₖ（Σpᵢ = 1），则各类次数 (X₁, ..., Xₖ) 的联合 PMF：

$$P(X_1=n_1, ..., X_k=n_k) = \frac{n!}{n_1! \cdots n_k!} p_1^{n_1} \cdots p_k^{n_k}$$

其中 Σnᵢ = n。

**重要性质**：
- 边缘分布：Xᵢ ~ B(n, pᵢ)
- 任意子集合并后仍服从多项分布（减少类别数）
- (Xᵢ, Xⱼ) 不独立（因为 Xᵢ + Xⱼ ≤ n 的约束）

### 9.2 真题完整解答：家教中心问题（第六题）

**题目背景**：N 位同学分四个年级（N₁, N₂, N₃, N₄），每天随机抽一人服务，连续 n 天。

**(1) P(X₁=30, X₂=20, X₃=20, X₄=30)**

每天抽取是独立的（有放回），每次抽到第 i 年级的概率 pᵢ = Nᵢ/N。  
(X₁, X₂, X₃, X₄) ~ Multinomial(n, p₁, p₂, p₃, p₄)，其中 n = 30+20+20+30 = 100。

$$P = \frac{100!}{30!\, 20!\, 20!\, 30!} \left(\frac{N_1}{N}\right)^{30} \left(\frac{N_2}{N}\right)^{20} \left(\frac{N_3}{N}\right)^{20} \left(\frac{N_4}{N}\right)^{30}$$

**(2) (X₁, X₂) 的边缘分布**

方法：将第三、四年级合并为"非一二年级"，概率 p₃₄ = (N₃+N₄)/N。

(X₁, X₂, X₃+X₄) ~ Multinomial(n, p₁, p₂, p₃₄)

所以：

$$P(X_1=k_1, X_2=k_2) = \frac{n!}{k_1!\, k_2!\, (n-k_1-k_2)!} p_1^{k_1} p_2^{k_2} (1-p_1-p_2)^{n-k_1-k_2}$$

其中 k₁ + k₂ ≤ n，k₁, k₂ ≥ 0。

**(3) 无放回抽样，Y = 抽到第 r 个大一同学时的总天数**

这是**负超几何分布**的模型。

设总共 N 人，其中大一 N₁ 人，非大一 N − N₁ 人。每天抽一人（不放回），Y 是第 r 个大一同学被抽到时的总天数。

Y 的值域：r ≤ Y ≤ N − N₁ + r。

P(Y = m) 的含义：前 m−1 天恰好抽到 r−1 个大一同学，第 m 天抽到第 r 个大一同学。

$$P(Y=m) = \frac{\binom{N_1}{r-1}\binom{N-N_1}{m-r}}{\binom{N}{m-1}} \cdot \frac{N_1 - (r-1)}{N - (m-1)}$$

解读：前 m−1 天从 N₁ 个大一中选 r−1 个、从 N−N₁ 个非大一中选 m−r 个；第 m 天从剩余中选中大一同学的概率。

化简：

$$P(Y=m) = \frac{\binom{N_1}{r}\binom{N-N_1}{m-r}}{\binom{N}{m}} \cdot \frac{r}{m}$$

## 第十部分：泊松分解定理

### 10.1 定理陈述与证明（真题第七题）

**定理**：N ~ P(λ)，每个个体独立以概率 p 被分类为 A 类（概率 1−p 为 B 类），则：
- X（A 类数量）~ P(λp)
- Y（B 类数量）~ P(λ(1−p))
- X, Y 相互独立

**证明**（需要完整掌握）：

计算 P(X=i, Y=j)：

$$P(X=i, Y=j) = P(X=i, Y=j, N=i+j) = P(X=i, Y=j \mid N=i+j) \cdot P(N=i+j)$$

给定 N = i+j，X|N ~ B(i+j, p)，所以：

$$P(X=i, Y=j \mid N=i+j) = \binom{i+j}{i} p^i (1-p)^j$$

$$P(N=i+j) = \frac{e^{-\lambda} \lambda^{i+j}}{(i+j)!}$$

相乘：

$$P(X=i, Y=j) = \binom{i+j}{i} p^i (1-p)^j \cdot \frac{e^{-\lambda} \lambda^{i+j}}{(i+j)!}$$

$$= \frac{(i+j)!}{i!\, j!} \cdot \frac{e^{-\lambda} (λp)^i (\lambda(1-p))^j}{(i+j)!}$$

$$= \frac{e^{-\lambda} (\lambda p)^i (\lambda(1-p))^j}{i!\, j!}$$

$$= \frac{e^{-\lambda p} (\lambda p)^i}{i!} \cdot \frac{e^{-\lambda(1-p)} (\lambda(1-p))^j}{j!}$$

最后一步用了 e⁻λ = e⁻λp · e⁻λ(1−p)。

这正是 P(λp) 的 PMF 与 P(λ(1−p)) 的 PMF 的乘积，因此 X ~ P(λp)，Y ~ P(λ(1−p))，且 X ⊥ Y。∎

**(2) X 的分布**

由上述证明已知：**X ~ P(λp)**。

## 第十一部分：正态分布专题

### 11.1 真题完整解答：正态参数反求（第九题）

**题目**：X ~ N(μ, σ²)，P(X ≤ 70) = 0.5，P(X ≤ 60) = 0.25。已知 Φ(0.25) = 0.5987，Φ(−0.6745) = 0.25。

**(1) 求 μ 和 σ**

由 P(X ≤ 70) = 0.5：

$$\Phi\left(\frac{70 - \mu}{\sigma}\right) = 0.5 \implies \frac{70 - \mu}{\sigma} = 0 \implies \mu = 70$$

（因为 Φ(0) = 0.5，正态分布的对称性）

由 P(X ≤ 60) = 0.25：

$$\Phi\left(\frac{60 - 70}{\sigma}\right) = 0.25 \implies \frac{-10}{\sigma} = -0.6745$$

$$\sigma = \frac{10}{0.6745} \approx 14.83$$

**(2) 5 人中至少 2 人体重超过 60kg**

P(体重 > 60) = 1 − P(X ≤ 60) = 1 − 0.25 = 0.75。

设 Y = 5 人中体重超 60kg 的人数，Y ~ B(5, 0.75)。

$$P(Y \geq 2) = 1 - P(Y=0) - P(Y=1)$$

$$P(Y=0) = \binom{5}{0}(0.75)^0(0.25)^5 = (0.25)^5 = \frac{1}{1024}$$

$$P(Y=1) = \binom{5}{1}(0.75)^1(0.25)^4 = 5 \times 0.75 \times \frac{1}{256} = \frac{3.75}{256} = \frac{15}{1024}$$

$$P(Y \geq 2) = 1 - \frac{1}{1024} - \frac{15}{1024} = 1 - \frac{16}{1024} = 1 - \frac{1}{64} = \frac{63}{64}$$

## 第十二部分：递推方法与组合概率

### 12.1 真题完整解答：偶数次正面（第十题）

**题目**：有偏硬币 p = 0.25，连续抛 n 次，恰好偶数次正面（0 次算偶数）的概率？

**方法一：递推法**

设 Pₙ = n 次抛掷中偶数次正面的概率。

递推关系：考虑第 n 次的结果：
- 若第 n 次反面（概率 1−p）：前 n−1 次需偶数次正面，概率 Pₙ₋₁
- 若第 n 次正面（概率 p）：前 n−1 次需奇数次正面，概率 1 − Pₙ₋₁

$$P_n = (1-p) \cdot P_{n-1} + p \cdot (1 - P_{n-1}) = (1-2p) P_{n-1} + p$$

初始条件：P₀ = 1（0 次抛掷，0 次正面，0 是偶数）。

解递推方程：这是一阶线性递推 Pₙ = aP_{n-1} + b，其中 a = 1−2p，b = p。

特解（稳态）：P* = p/(1−(1−2p)) = p/(2p) = 1/2。

齐次解：Pₙ − 1/2 = (1−2p)ⁿ (P₀ − 1/2) = (1−2p)ⁿ · 1/2。

$$\boxed{P_n = \frac{1 + (1-2p)^n}{2}}$$

当 p = 0.25：

$$P_n = \frac{1 + (0.5)^n}{2} = \frac{1 + 1/2^n}{2} = \frac{2^n + 1}{2^{n+1}}$$

**方法二：二项式定理**

$$P_n = \sum_{k \text{ even}} \binom{n}{k} p^k (1-p)^{n-k}$$

利用恒等式：偶数项之和 = [(1−p+p)ⁿ + (1−p−p)ⁿ]/2 = [1 + (1−2p)ⁿ]/2。

> **验证**：n=1 时 P₁ = (1 + 0.5)/2 = 0.75 = 1−p ✓（只有 0 次正面算偶数）  
> n=2 时 P₂ = (1 + 0.25)/2 = 0.625 = (0.75)² + (0.25)² = 0.5625 + 0.0625 = 0.625 ✓

## 第十三部分：条件分布专题

### 13.1 离散型条件分布

$$P(X=x_i | Y=y_j) = \frac{P(X=x_i, Y=y_j)}{P(Y=y_j)}$$

### 13.2 连续型条件密度

$$f_{X|Y}(x|y) = \frac{f(x,y)}{f_Y(y)}, \quad f_Y(y) > 0$$

**条件密度的 Bayes 形式**：

$$f_{Y|X}(y|x) = \frac{f_{X|Y}(x|y) \cdot f_Y(y)}{f_X(x)}$$

这是 Bayes 公式在连续型随机变量上的推广。

**条件概率的全概率公式**：

$$f_X(x) = \int_{-\infty}^{\infty} f_{X|Y}(x|y) \cdot f_Y(y) \, dy$$

> **解题技巧**：条件密度中的条件变量可以视为固定常数进行计算。例如 f_{X|Y}(x|y) 中，y 是固定的，只需关注 x 的函数形式。

### 13.3 独立性的多种判定方法

1. **联合 CDF 分解**：F(x,y) = Fₓ(x)·F_Y(y)
2. **联合 PDF 分解**：f(x,y) = fₓ(x)·f_Y(y)（+ 矩形区域）
3. **条件 = 边缘**：f_{X|Y}(x|y) = fₓ(x)（条件分布不依赖于条件变量）
4. **联合 PMF 分解**：P(X=xᵢ, Y=yⱼ) = P(X=xᵢ)·P(Y=yⱼ)

> **快速判定技巧**：拿到联合 PDF 后，先看定义域是否可分离（矩形/独立范围），再看函数表达式是否可分解为 g(x)·h(y)。两者必须同时满足。

## 第十四部分：解题方法论总结

### 14.1 判断题策略

1. **反例优先**：遇到任何"看起来对"的命题，先试着构造反例。常用的反例来源：不可列集合、泊松分布（无记忆性陷阱）、条件独立 vs 独立、连续但 PDF 不连续。

2. **量词敏感**："对任意"→ 一个反例即可否定；"存在"→ 需要对所有情况验证才能否定。

3. **充分 vs 必要**：很多命题是某定理的逆命题或弱化版，检查条件是否充分/必要。

### 14.2 各题型标准解法

| 题型 | 标准方法 | 关键步骤 |
|------|----------|----------|
| Bayes 公式题 | 确定划分 → 先验 → 似然 → 套公式 | 不要忘记全概率公式作为分母 |
| 变量函数分布 | CDF 法 → 化简不等式 → 求导 | 注意值域和单调性方向 |
| 联合分布问题 | F(∞,∞)=1 求常数 → 偏导得 PDF → 积分得边缘 | 验证独立性看分离+矩形 |
| 正态参数反求 | Φ(0)=0.5 定 μ → 查表定 σ | 注意 Φ(−z) = 1−Φ(z) |
| 递推概率 | 建立 Pₙ 关于 Pₙ₋₁ 的递推式 → 解线性递推 | 特解 + 齐次解 |
| 多项分布 | 识别独立重复试验 → 套多项分布公式 | 边缘分布通过合并类别获得 |
| 事件域/极限 | 逐步取补取并 / 套 lim sup/inf 定义 | 画图辅助理解 |

### 14.3 高频易错清单

1. **互斥 ≠ 独立**：互斥事件 A∩B = ∅ 时，若 P(A)>0, P(B)>0，则一定不独立（因为 P(A∩B) = 0 ≠ P(A)P(B)）。
2. **泊松无记忆性**：泊松分布**没有**无记忆性，只有几何分布和指数分布有。
3. **CDF 法忘写值域**：f_Y(y) 必须注明 y 的范围，否则扣分。
4. **混合分布中 P(X ≥ a) 与 P(X > a) 的区别**：连续点上相等，跳跃点上差 P(X=a)。
5. **边缘正态 ⇏ 联合正态**：两个边缘都是正态的随机变量，联合分布不一定是二元正态。
6. **独立性的区域条件**：f(x,y) = xy 在三角形区域上不独立，即使函数可分离。
7. **条件独立 vs 独立**：互不蕴含，需要分别验证。
8. **Φ 表查值方向**：P(X ≤ a) = Φ((a−μ)/σ)，注意分子的正负号。
